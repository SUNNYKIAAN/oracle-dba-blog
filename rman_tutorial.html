RMAN Backup Tutorial

RMAN Backup Best Practices

Posted on July 26, 2025

RMAN (Recovery Manager) is Oracle's recommended tool for backup and recovery. In this tutorial, we'll cover best practices and commonly used RMAN commands to protect your Oracle database effectively.

1. Configure RMAN Settings

RMAN> CONFIGURE RETENTION POLICY TO REDUNDANCY 2;
RMAN> CONFIGURE CONTROLFILE AUTOBACKUP ON;

2. Full Database Backup

RMAN> BACKUP DATABASE PLUS ARCHIVELOG;

3. Backup Validation

RMAN> RESTORE DATABASE VALIDATE;

4. Deleting Obsolete Backups

RMAN> DELETE OBSOLETE;

5. Scheduling Backups with Crontab

You can schedule RMAN scripts via crontab using shell scripts that call `rman target / @backup_script.rcv`.
